Vestix — Laravel Forge setup checklist
========================================

Server (Forge → Create Server)
  [ ] PHP 8.3
  [ ] MySQL 8
  [ ] Node.js enabled
  [ ] UFW / fail2ban defaults

Site (Forge → Create Site)
  [ ] Domain configured
  [ ] Web directory: /public
  [ ] Database created (note credentials)
  [ ] SSL: Let's Encrypt (DNS A-record must point to server first)
  [ ] Git: davyvanbokhoven/swng.trade, branch main
  [ ] Deploy on push: optional

Environment (Forge → Site → Environment)
  [ ] Copy forge/production.env.example as starting point
  [ ] Set APP_KEY (php artisan key:generate --show)
  [ ] Set APP_URL to https://your-domain
  [ ] Fill DB_*, API keys, mail, PHP_BINARY

Deployment (Forge → Site → Deployment)
  [ ] Paste contents of forge/deploy.sh into Deploy Script
  [ ] Run first deploy (Deploy Now)

Scheduler (Forge → Site → Scheduler)
  [ ] Cron active: * * * * * php artisan schedule:run
  [ ] Verify: php artisan schedule:list

First admin (SSH)
  [ ] php artisan vestix:create-user you@example.com --name="You" --password="..." --squad="Alpha Squad"
  [ ] Or register at /admin/register

Post-deploy (SSH)
  [ ] php artisan vestix:smoke-test
